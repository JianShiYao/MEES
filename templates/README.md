# Templates

用于存放可直接复制到项目中的治理、需求、设计、测试、安全、发布和评审模板。

v0.4.0 的 18 项模板（P0 13 项、P1 5 项）已通过 V4-G5，形成仓库内部模板基线。建设范围见 [v0.4.0 工程模板建设计划](../docs/00_Introduction/05_v0.4工程模板建设计划.md)，复制、裁剪和证据规则见 [v0.4 模板目录与使用规则](../docs/13_Templates/v0.4模板目录与使用规则.md)。

- `templates/` 保存产品无关的规范模板。
- `docs/13_Templates/` 保存模板目录、选择指南和字段说明。
- `docs/15_Case_Study/` 保存脱敏项目实例和使用反馈。
- 空模板属于 `D` 类定义证据；只有经真实项目确认的实例才属于 `P` 类证据。
- v0.4 的 18 项模板和 v0.5.1 的两项 ESS 增量模板状态为“已批准（内部基线）”；这不等同于真实项目部署证据、独立评估或认证。
- `TPL-V05-*` 属于 v0.5 增量，仍按各自头部状态管理，不计入 v0.4 的 18 项冻结数量。
