# MEES 首页

欢迎进入 **MEES — 现代嵌入式研发工程体系**。

## 使用路径

- 管理者：先阅读 [MEES 总览](00_Introduction/00_MEES总览.md) 和 [建设路线图](00_Introduction/01_建设路线图.md)。
- 项目经理：阅读项目管理、风险、质量策划与度量。
- 系统工程师：阅读系统需求、系统架构、系统验证。
- 软件工程师：阅读软件需求、架构、详细设计、编码与单元验证。
- 测试工程师：阅读集成测试、系统测试、确认测试与自动化测试。
- 质量人员：阅读 ASPICE、[ISO/IEC 33020 中文工程解读](11_Process_Management/ISO_IEC_33020中文工程解读.md)、审核、配置管理和度量。
