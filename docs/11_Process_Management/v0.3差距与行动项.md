# v0.3 差距与行动项

> 文档编号：MEES-LIF-009
> 版本：v0.3.0
> 状态：已批准（内部人工确认）
> 所有者：工程过程负责人
> 最后更新：2026-07-14

## 1. 目的

汇总 v0.3 五类标准映射发现的过程、工作产品、角色和实施证据差距，为 V3-G3/V3-G5 评审提供唯一行动台账，并形成 v0.4 工程模板的范围、字段和优先级输入。

差距存在不等于 v0.3 映射失败；只要映射边界诚实、责任明确且不阻断文档基线使用，可以在后续版本关闭。任何真实符合性或能力结论仍必须等待适用差距和项目证据关闭。

## 2. 优先级

| 优先级 | 判定 |
|---|---|
| P0 | 影响安全/网络安全、唯一责任、发布控制或 v0.4 核心模板范围，优先建设 |
| P1 | 影响过程稳定部署、能力属性或组织治理，应在 v0.4–v1.0 关闭 |
| P2 | 生命周期后端或优化类接口，可在边界受控时后续关闭 |
