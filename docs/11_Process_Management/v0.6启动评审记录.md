## 5. 人工决定

**V6-G0：Go。**

- 批准 v0.6 建设计划中的 WP0–WP7、权限等级、证据边界、M6 验收和 V6-G0–G6 门禁。
- 批准进入 WP1，按 F4、F5、F6 的受控关闭判据建设可信自动化输入。
- 允许并行细化 WP2 公共协议草案，但六类 Agent 最小切片须在 V6-G1 后进入实现和在 V6-G2 冻结。
- O2 只读、O3 受控写入、BMS 案例和 v0.6 发布分别受 V6-G3、V6-G4、V6-G5、V6-G6 控制。
- 不允许把本批准表述为 Agent 已可用、Obsidian MCP 已连接、真实独立评估、认证或产品批准。
