# ISO 9001 过程映射

> 文档编号：MEES-LIF-007
> 版本：v0.3.0
> 状态：已批准（内部人工确认）
> 所有者：质量负责人
> 最后更新：2026-07-14

> 本文是 MEES 基于 ISO 9001:2015 和 Amd 1:2024 形成的原创工程接口映射，不是标准译文、符合性声明或质量管理体系认证结论。

## 1. 目的与范围

把 ISO 9001 的组织环境、领导、策划、支持、运行、绩效评价和改进关注点映射到 MEES 工程过程、门禁、工作产品和证据，并识别工程体系之外仍需由组织级 QMS 承接的责任。

来源和版权边界继承 [v0.3 标准映射总览](../11_Process_Management/v0.3标准映射总览.md) 的 `SRC-QMS-001`。v0.3 使用 ISO 9001:2015 + Amd 1:2024；后续版本正式发布前只做变化观察。

## 2. 适用性与责任边界

- MEES 是嵌入式研发过程体系，不替代完整组织 QMS。
- 产品、项目、系统、软件、测试、配置和发布负责人保留既有工程责任；质量负责人维护质量策划、评价、不符合与改进接口。
- 组织治理、财务、人力资源、采购、生产、服务和认证审核不在 MEES 核心范围时，必须记录外部责任和接口。
- 本文证据为 `D` 或模拟走查 `D+S`，不能据此声明 ISO 9001 符合性或认证状态。
