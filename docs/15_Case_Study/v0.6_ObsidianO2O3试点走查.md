## 2. 前置条件（在你的机器上）

1. Obsidian 安装并启用社区插件 **Local REST API**，复制 API Key。
2. **REST 基址用去掉 `/mcp/` 的根**：`MEES_OBSIDIAN_URL=https://127.0.0.1:27124`（`/mcp/` 端点是给 MCP 客户端的，非本适配器）。设环境变量（仅 shell，勿提交）：`MEES_OBSIDIAN_URL`、`MEES_OBSIDIAN_TOKEN`。
3. **自签证书（A 方案，正常 TLS 校验）**：在插件设置下载证书，存为 `config/obsidian-cert.pem`（已 Git 忽略），把 `config/obsidian_integration.json` 的 `remote_ca_cert` 设为 `config/obsidian-cert.pem`。未配置证书时对自签 HTTPS 会按预期报 TLS 校验失败。
4. 确认状态：`python scripts/obsidian_adapter.py status` → `connected: true`。
5. 审计输出：`build/obsidian/audit.jsonl`（Git 忽略；只记 sha256 摘要，不记 token/正文）。安全：API Key 仅存环境变量；试点后建议在插件重新生成。
