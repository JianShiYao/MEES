# 储能功能安全管理（FSM）过程

> 文档编号：MEES-PRO-701
> 版本：v0.5.1-dev
> 状态：评审中（v0.5.1 收口候选）
> 所有者：功能安全负责人
> 最后更新：2026-07-15

> 本文是 MEES 面向固定式储能/电池系统的原创功能安全管理过程，基于 IEC 61508 的功能安全管理理念，不是标准译文、符合性声明或安全认证结论。

## 1. 目的

定义固定式储能/电池系统功能安全生命周期的管理方式：范围、角色、独立性、能力、裁剪、修改控制与安全生命周期接口，为 [储能功能安全映射 MEES-MAP-001](储能功能安全_IEC61508_UL9540A映射.md) 提供治理载体，关闭 `GAP-V03-ESS-001` 的结构部分。

## 2. 适用范围

仅当项目属于固定式储能/电池系统范围，或合同/法规要求采用等效功能安全方法时启用。安全完整性以 **SIL（IEC 61508）** 表达，不使用汽车 ASIL。硬件/电化学层安全按 `EXT-HW` 外部受控对象承接。

## 3. 安全生命周期接口

FSM 不新建平行生命周期，而是把安全活动挂接到 MEES G0–G6：

| 门禁 | 安全活动接口 |
|---|---|
| G0/G2 | 危害分析、SIL 确定、安全需求与安全功能定义（`HAZ→SIL→SAFE-FN`） |
| G3 | 安全架构、机制、HFT/SFF/DC 论证 |
| G4 | 安全相关实现与单元验证，UL 9540A 测试计划就绪 |
| G5 | 安全验证、确认与 UL 9540A 结果汇总 |
| G6 | 安全发布建议、遗留偏差、独立 FSA 结论 |

## 4. 角色、独立性与能力

| 角色 | 职责 | 独立性 |
|---|---|---|
| 功能安全负责人 | 拥有安全计划、危害分析、SIL、安全论证与 FSA 结论 | 对安全结论独立于交付进度压力 |
| 系统/软件负责人 | 拥有安全需求分配、安全架构与安全功能实现 | — |
| 测试负责人 | 拥有安全验证、UL 9540A 测试执行与证据 | 验证独立于实现 |
| 独立评估人 | 按 SIL 执行独立功能安全评估（FSA） | 与项目开发线独立 |

角色名沿用[核心过程总览](../01_Main_Process/00_核心过程总览.md)权威表；"功能安全负责人"为安全治理唯一领导角色。

## 5. 活动

1. 建立安全计划：范围、SIL 目标假设、活动、里程碑与确认措施。
2. 组织危害分析与 SIL 确定（热失控为核心），产出 `HAZ` 与 `SIL`（模板 `TPL-V05-001`）。
3. 定义安全功能 `SAFE-FN` 与安全状态，分配到系统/软件与 `EXT-HW`。
4. 协调安全需求、安全架构与安全机制评审。
5. 建立 UL 9540A 热失控测试计划与证据接口（模板 `TPL-V05-002`）。
6. 汇总安全验证/确认，形成安全案例与安全发布建议。
7. 在修改与问题流程下控制安全影响，触发危害与 SIL 重评。
8. 组织独立 FSA，记录独立性与结论。

## 6. 工作产品

| 工作产品 | 最小要求 |
|---|---|
| 安全计划 | 范围、SIL 假设、活动、里程碑、确认措施、独立性 |
| 危害分析与 SIL 记录 | `HAZ`、风险评估、`SIL` 目标、安全功能（`TPL-V05-001`） |
| 安全需求与追溯 | `SAFE-FN`→安全需求→设计→验证的双向追溯 |
| UL 9540A 证据接口 | 层级测试计划、结果引用、缓解设计输入（`TPL-V05-002`） |
| 安全案例与发布建议 | 论证、遗留偏差、FSA 结论、发布适宜性 |

## 7. 修改控制

安全相关修改进入[变更与问题管理过程](../01_Main_Process/07_变更与问题管理过程.md)，必须评估对危害、SIL、安全功能、验证与安全案例的影响；`EXT-HW` 变更须由供方安全证据与验收支撑。

## 8. 裁剪规则

- 非储能产品整体不适用本过程，记录理由与批准人。
- 储能产品不得裁剪危害分析、SIL 确定、安全验证与独立 FSA；低 SIL 项目可简化独立性层级但须记录依据。

## 9. 证据

- 安全计划、危害/SIL 记录、安全需求追溯。
- UL 9540A 测试计划/结果引用与缓解设计输入。
- 安全案例、独立 FSA 记录与安全发布建议。
- 本过程为 `D` 级文档定义；无真实 SIL 决策、UL 9540A 结果或安全案例证据前，不声明功能安全符合性。

## 10. 标准映射

| 标准 | 映射说明 |
|---|---|
| IEC 61508-1 | 功能安全管理、生命周期、能力与 FSA 接口 |
| UL 9540A | 热失控测试证据接口（`TPL-V05-002`、`MAP-ESS-003`） |
| UL 1973 / IEC 62619 | 电池/电芯安全经 `EXT-HW` 承接 |
